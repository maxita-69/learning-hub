Lezione 27 - Corso JavaScript

In questo capitolo ci concentreremo sulla *Coercizione Esplicita*, ovvero quel processo in cui siamo noi programmatori a richiedere intenzionalmente al motore JavaScript di trasformare un dato da un tipo a un altro2.

*1\. Conversione Esplicita a Number*

Per convertire manualmente un valore in un numero, utilizziamo la funzione nativa Number(). Passiamo come parametro l'elemento da convertire e JavaScript applicherà le sue regole di trasformazione3.

Analizziamo i risultati di queste conversioni, inclusi i casi particolari:

JavaScript

// Conversione di stringhe numeriche

Number("2"); // 2

// Conversione della stringa vuota

Number(""); // 0

// Una stringa vuota viene trattata come valore nullo/zero.

// Conversione di null e booleani

Number(null); // 0

Number(false); // 0

Number(true); // 1

// true diventa 1, mentre false e null diventano 0\[cite: 5, 6\].

// Conversione di stringhe non numeriche

Number("stringa"); // NaN

// Non è possibile trasformare una parola in numero.

// Il risultato è NaN (Not a Number)\[cite: 6\].

*Scorciatoia: L'Operatore Unario +*

Esiste un metodo molto diffuso e veloce per convertire una stringa in numero senza digitare l'intera funzione Number(): l'utilizzo dell'operatore *più (+) come operatore unario*.

Basta anteporre il simbolo + alla stringa (o al valore) che vogliamo convertire. Il comportamento è identico alla funzione Number()4.

JavaScript

// Operatore Unario +

+"10"; // 10

+"ciao"; // NaN

// Anche qui, se la stringa non è numerica, la conversione fallisce\[cite: 13\].

*2\. Conversione Esplicita a String*

Quando abbiamo bisogno di trasformare un valore nel suo equivalente testuale, utilizziamo la funzione String(). Questa operazione è generalmente più intuitiva rispetto a quella numerica, poiché si limita a rappresentare il valore sotto forma di caratteri5.

JavaScript

String(2); // "2"

String(null); // "null"

String(true); // "true"

*3\. Conversione Esplicita a Boolean*

La conversione verso il tipo booleano è fondamentale per la gestione della logica condizionale. Possiamo effettuarla utilizzando la funzione Boolean().

Per capire come funziona, dobbiamo ricordare che in JavaScript esiste un insieme ristretto di valori che sono intrinsecamente falsi (*Falsy values). Tutto ciò che non rientra in questa lista è considerato vero (Truthy values*)6.

*I Valori Falsy (False)*

Se passiamo uno dei seguenti valori alla funzione Boolean(), otterremo sempre false:

7

JavaScript

Boolean(""); // false

Boolean(NaN); // false

Boolean(undefined); // false

Boolean(0); // false

*I Valori Truthy (True)*

Tutto il resto, in JavaScript, è considerato true. Questo porta a casi che possono confondere i principianti, come la stringa "0".

*Attenzione:* La stringa "0" non è vuota, contiene un carattere. Pertanto, convertita in booleano, risulta vera.

JavaScript

// Numeri diversi da zero

Boolean(1); // true

// Stringhe non vuote (anche se contengono "0")

Boolean("0"); // true

// Attenzione: "0" è una stringa piena, quindi è true\! \[cite: 10\]

*Scorciatoia: Il Doppio Punto Esclamativo \!\!*

Così come esiste l'operatore unario + per i numeri, esiste una sintassi rapida per i booleani: il doppio punto esclamativo (\!\!).

Il primo \! nega il valore (convertendolo implicitamente), il secondo \! lo nega di nuovo, riportandolo al suo senso logico originale ma sotto forma di booleano puro8.

JavaScript

\!\!10; // true

\!\!0; // false
