*Corso di JavaScript - Lezione: L'Oggetto String e i suoi Metodi*

*1\. Introduzione all'Oggetto String*

Ho già pronunciato la frase che in JavaScript "gli oggetti sono il re". Quindi non ci sorprenderà il fatto che disponiamo di un oggetto predefinito String, tramite il quale è possibile creare una stringa proprio come è fatto con Number.

Creazione di una stringa come oggetto:

let str = new String("Ciao");

console.log(typeof str); // "object"

Facciamo console.log di String e diamo un'occhiata a cosa contiene:

javascript

console.log(String); // ƒ String() { \[native code\] }

Dice sempre che è una funzione, ma noi sappiamo che le funzioni in JavaScript sono oggetti.

*2\. Metodi negli Oggetti*

Quando utilizziamo new viene creato un oggetto che può contenere dei metodi.

Esempio di metodo in un oggetto:

let oggetto = {  
    fn: function() {  
        return "ciao";  
    }  
};

console.log(oggetto.fn()); // "ciao"

Invocare un metodo con la sintassi nomeOggetto.nomeMetodo() ormai è una procedura che conosciamo.

*3\. Il Caso Particolare delle Stringhe*

La novità è che quando abbiamo bisogno di conoscere la lunghezza di una stringa, scriviamo:

let a = "Ciao";

console.log(a.length); // 4

Cerchiamo di accedere alla proprietà length non invocata su un oggetto ma su un tipo primitivo. Questo effettivamente può risultare strano.

Estrazione di sottostringa:

javascript

let a = "Limitiamoci a dieci caratteri";  
let b = a.substr(0, 10);  
console.log(b); // "Limitiamoc"

console.log(b.length); // 10

Anche in questo caso accediamo ad un metodo, ma non attraverso un oggetto ma attraverso un dato primitivo.

*4\. Ereditarietà Prototipale*

Il motivo per cui riusciamo a farlo è che internamente il tipo primitivo viene convertito in oggetto, e ogni oggetto in JavaScript eredita in modo automatico una serie di proprietà e metodi a cui quindi può accedere.

Esempio:

let object = {};

console.log(object.toString()); // "\[object Object\]"

Noi vediamo che object effettivamente non contiene toString, ma siamo comunque riusciti ad accedere al metodo. Questo succede perché ogni oggetto accede al sistema di ereditarietà prototipale che studieremo più avanti nel corso.

*5\. Metodi Pratici con le Stringhe*

Procediamo ed analizziamo alcuni metodi che possiamo utilizzare con le stringhe. Lo facciamo direttamente nella console per comodità.

Trasformazione case:

"ciao".toUpperCase(); // "CIAO"

"OK".toLowerCase(); // "ok"

Ricerca di caratteri:

javascript

"Gianluca".indexOf('a'); // 2

Infatti la G è in posizione zero, la i è in posizione uno e la a in posizione due.

javascript

"Gianluca".lastIndexOf('a'); // 7

La stringa "Gianluca" è composta da otto caratteri, quindi la a finale in posizione sette.

Verifica presenza sottostringa:

javascript

"Mi chiamo Gianluca".includes("mi"); // true  
"Mi chiamo Gianluca".includes("chi"); // true

"Mi chiamo Gianluca".includes("foo"); // false

A differenza di indexOf che restituisce la posizione, il metodo includes restituisce soltanto true oppure false.

Suddivisione stringa:

javascript

"Questa è una frase".split(" "); // \["Questa", "è", "una", "frase"\]

Il risultato è un array con le varie parti della stringa.

Ripetizione stringa:

javascript

"ciao ".repeat(5); // "ciao ciao ciao ciao ciao "

Eliminazione spazi:

javascript

"   ciao   ".trim(); // "ciao"

Il metodo trim va ad eliminare gli spazi all'inizio e alla fine della stringa.
