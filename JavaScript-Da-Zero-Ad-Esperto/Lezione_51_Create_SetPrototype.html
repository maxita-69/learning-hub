# *Metodi per la Gestione dei Prototipi e la Parola Chiave this*

*Metodi per Impostare il Prototipo*

*Metodo 1: Object.create()*

Definiamo un oggetto obj1 con una serie di proprietà:

const obj1 = {

p1: 10,

p2: 20,

p3: 30

};

Adesso vogliamo che un nuovo oggetto abbia come prototipo obj1. Per farlo utilizziamo Object.create(), passando come parametro l'oggetto obj1:

const obj2 = Object.create(obj1);

Vediamo se il prototipo di obj2 è proprio obj1:

console.log(Object.getPrototypeOf(obj2) === obj1);

// Output: true

Adesso se vogliamo aggiungere delle proprietà a obj2 lo possiamo tranquillamente fare:

obj2.p4 = 40;

obj2.p5 = 50;

Questa procedura però di *aggiunta successiva delle proprietà è abbastanza scomoda*.

*Metodo 2: Object.setPrototypeOf()*

Per far sì che un oggetto abbia come prototipo un altro oggetto possiamo anche utilizzare il metodo *Object.setPrototypeOf()*.

Definiamo un oggetto obj2 con alcune proprietà:

const obj2 = {

p4: 40,

p5: 50

};

Invochiamo il metodo setPrototypeOf() sulla Constructor Function Object, passando come primo parametro obj2 e come secondo parametro l'oggetto che vogliamo sia il prototipo di obj2:

Object.setPrototypeOf(obj2, obj1);

Controlliamo se abbiamo fatto bene. Vediamo se il prototipo di obj2 è proprio obj1:

console.log(Object.getPrototypeOf(obj2) === obj1);

// Output: true

*⚠️ ATTENZIONE: Abbiamo detto che la proprietà __proto__ è una proprietà deprecata. Per conoscere il prototipo di un oggetto dobbiamo utilizzare il metodo Object.getPrototypeOf()*.

*Object.getPrototypeOf() - Metodo Standard*

Ad esempio, vogliamo conoscere il prototipo di obj2, lo passiamo come parametro:

const protoObj2 = Object.getPrototypeOf(obj2);

console.log(protoObj2);

// Output: obj1

Vogliamo conoscere il prototipo di obj1, facciamo la stessa cosa:

const protoObj1 = Object.getPrototypeOf(obj1);

console.log(protoObj1);

// Output: Object.prototype
