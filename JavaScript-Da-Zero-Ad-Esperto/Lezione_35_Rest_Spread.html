*4\. Rest Parameters (...)*

Il *Rest Parameter permette di raccogliere un numero indefinito di argomenti in un vero e proprio Array*.

*Sintassi*

Si utilizzano tre puntini (...) seguiti dal nome che vogliamo dare all'array (es. ...numeri). Deve essere sempre l'*ultimo* parametro nella definizione.

*Esempio: Somma Dinamica*

Riscriviamo la funzione calcola per accettare infiniti numeri e sommarli.

JavaScript

function calcola(...numeri) {

// 'numeri' è un vero Array \[10, 20, 30, ...\]

console.log(numeri);

let somma = 0;

// Possiamo usare for...of o for classico

for (let numero of numeri) {

somma += numero;

}

console.log(somma);

}

calcola(10, 20, 30, 40);

// Output array: \[10, 20, 30, 40\]

// Output somma: 100

*Combinare Parametri Fissi e Rest*

Possiamo definire alcuni parametri fissi iniziali e raccogliere "il resto" in un array.

JavaScript

// param1 prenderà il primo valore

// param2 prenderà il secondo

// ...numeri raccoglierà tutti gli altri dal terzo in poi

function calcola(param1, param2, ...numeri) {

console.log(param1);

console.log(param2);

console.log(numeri);

}

calcola(10, 20, 30, 40, 1, 2, 10);

*Risultato visualizzato:*

  • param1: 10
  • param2: 20
  • numeri: \[30, 40, 1, 2, 10\]

*5\. Spread Operator (...)*

Lo *Spread Operator utilizza la stessa sintassi (...) del Rest, ma agisce in modo opposto: invece di raggruppare, espande* (o "spalma") un array nei suoi singoli elementi.

*Caso d'uso 1: Passare un Array a funzioni che richiedono argomenti singoli*

Supponiamo di avere un array di numeri e di voler trovare il minimo. Math.min(arr) non funzionerebbe perché si aspetta numeri, non un array.

JavaScript

let arr = \[10, 20, 30, 1, 4\];

// ERRORE Logico: Math.min(\[10, 20...\]) -> NaN

// SOLUZIONE con Spread Operator

// Espande l'array in: 10, 20, 30, 1, 4

let minimo = Math.min(...arr);

console.log(minimo);

// Output: 1

*Caso d'uso 2: Unire (Merge) Array*

Lo Spread Operator è potentissimo per combinare array o crearne copie.

JavaScript

let a = \[1, 2, 3\];

let b = \[4, 5, 6\];

// Creiamo un nuovo array 'c' espandendo 'a' e 'b' al suo interno

let c = \[...a, ...b\];

console.log(c);

// Output: \[1, 2, 3, 4, 5, 6\]
