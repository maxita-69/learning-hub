*Metodi Standard per Accedere al Prototipo*

Anche se __proto__ è utile per comprendere il funzionamento della catena dei prototipi, in codice di produzione è preferibile utilizzare i metodi standard:

*Object.getPrototypeOf()*

// Metodo standard per ottenere il prototipo

const proto = Object.getPrototypeOf(Ferrari458);

console.log(proto === Ferrari.prototype);

// Output: true

*Object.setPrototypeOf()*

// Metodo per impostare il prototipo (sconsigliato per performance)

const newObj = {};

Object.setPrototypeOf(newObj, Ferrari.prototype);

*⚠️ NOTA:* Object.setPrototypeOf() può causare problemi di performance. È preferibile utilizzare Object.create() al momento della creazione dell'oggetto.

*Verifica dell'Appartenenza alla Catena*

*instanceof*

L'operatore instanceof verifica se un oggetto ha nella sua catena dei prototipi il prototype di una Constructor Function:

console.log(Ferrari458 instanceof Ferrari);

// Output: true

console.log(Ferrari458 instanceof Object);

// Output: true

console.log(Ferrari458Special instanceof Ferrari);

// Output: true

*isPrototypeOf()*

Metodo per verificare se un oggetto esiste nella catena dei prototipi di un altro oggetto:

console.log(Ferrari.prototype.isPrototypeOf(Ferrari458));

// Output: true

console.log(Ferrari458.isPrototypeOf(Ferrari458Special));

// Output: true

console.log(Object.prototype.isPrototypeOf(Ferrari458));

// Output: true

*Proprietà Proprie vs Proprietà Ereditate*

*hasOwnProperty()*

Questo metodo (ereditato da Object.prototype) permette di distinguere tra proprietà proprie dell'oggetto e proprietà ereditate:

Ferrari458.Modello = "458 Italia";

console.log(Ferrari458.hasOwnProperty("Modello"));

// Output: true (proprietà propria)

console.log(Ferrari458.hasOwnProperty("Marchio"));

// Output: false (proprietà ereditata da Ferrari.prototype)

console.log(Ferrari458.hasOwnProperty("toString"));

// Output: false (proprietà ereditata da Object.prototype)

*Visualizzazione della Catena dei Prototipi*

Una funzione utile per visualizzare l'intera catena dei prototipi:

function visualizzaCatenaPrototipi(obj) {

let currentObj = obj;

let level = 0;

while (currentObj \!== null) {

console.log(\`Livello ${level}:\`, currentObj);

currentObj = Object.getPrototypeOf(currentObj);

level++;

}

console.log(\`Livello ${level}: null\`);

}

// Utilizzo

visualizzaCatenaPrototipi(Ferrari458Special);

*Implicazioni per le Performance*

  • *Accesso alle proprietà*: La ricerca lungo la catena dei prototipi ha un costo. Le proprietà più in alto nella catena richiedono più tempo per essere trovate.
  • *Modifica del prototipo*: Cambiare il prototipo di un oggetto esistente (con setPrototypeOf) può invalidare le ottimizzazioni del motore JavaScript.
  • *Best practice*: Definire la struttura dei prototipi all'inizio e mantenerla stabile durante l'esecuzione del programma.
