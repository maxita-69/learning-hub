*Lezione: Il Tipo Booleano e la Truthy/Falsy Evaluation*

*1\. Introduzione al Tipo Booleano*

Altro tipo di dato primitivo è il tipo booleano. Come ovviamente sappiamo, un valore booleano può essere true oppure false.

È un tipo di dato importantissimo, perché dal fatto che qualcosa sia vero oppure falso, l'azione conseguente può essere diametralmente opposta.

*2\. Assegnazione Esplicita di Valori Booleani*

Il più semplice esempio che possa fare è quando esplicitamente assegniamo true oppure false ad una variabile:

let isDeveloper = true;

if (isDeveloper) {  
    console.log("Developer");  
} else {  
    console.log("Altrimenti");

}

Risultato: "Developer"

Abbiamo ovviamente "Developer", in quanto la condizione all'interno dell'if è soddisfatta perché isDeveloper è true.

*3\. Valori Booleani come Risultato di Espressioni*

Nella maggior parte dei casi, però, un valore booleano sarà il risultato di un'espressione, ad esempio un confronto tra due numeri:

if (4 > 3) {  
    console.log("4 è maggiore di 3");  
} else {  
    console.log("Altrimenti");

}

Risultato: "4 è maggiore di 3"

Confronto tra stringhe:

javascript

let str1 = "ciao"; // tutto in minuscolo  
let str2 = "Ciao"; // con la C maiuscola

if (str1 === str2) {  
    console.log("Uguali");  
} else {  
    console.log("No");

}

Risultato: "No"

Perché JavaScript è case-sensitive, quindi la stringa "ciao" tutto in minuscolo è diverso dalla stringa "Ciao" con la C in maiuscolo.

*4\. Valutazione Booleana delle Variabili*

Anche una singola variabile all'interno di un if verrà valutata come true oppure false:

let A = 10;  
let B = 0;  
let C = "ciao";

if (A) {  
    console.log("A è vero");  
} else {  
    console.log("A è falso");  
}  
// Risultato: "A è vero"

if (B) {  
    console.log("B è vero");  
} else {  
    console.log("B è falso");  
}  
// Risultato: "B è falso"

if (C) {  
    console.log("C è vero");  
} else {  
    console.log("C è falso");  
}

// Risultato: "C è vero"

*5\. Valori Falsy in JavaScript*

In JavaScript ci sono dei valori che vengono considerati false. Questi valori sono:

  • Il booleano false
  • La stringa vuota ""
  • Lo zero 0
  • Meno zero -0
  • NaN (Not a Number)
  • null
  • undefined

Esempi pratici:

if ("") {  
    console.log("Vero");  
} else {  
    console.log("Falso");  
}  
// Risultato: "Falso"

if (0) {  
    console.log("Vero");  
} else {  
    console.log("Falso");  
}  
// Risultato: "Falso"

if (-0) {  
    console.log("Vero");  
} else {  
    console.log("Falso");  
}  
// Risultato: "Falso"

if (NaN) {  
    console.log("Vero");  
} else {  
    console.log("Falso");  
}  
// Risultato: "Falso"

if (null) {  
    console.log("Vero");  
} else {  
    console.log("Falso");  
}  
// Risultato: "Falso"

if (undefined) {  
    console.log("Vero");  
} else {  
    console.log("Falso");  
}

// Risultato: "Falso"

*6\. Valori Truthy in JavaScript*

Tutti gli altri valori in JavaScript sono considerati true:

if ("0") {          // Vero - stringa "0"  
if (" ") {          // Vero - stringa con spazio  
if ({}) {           // Vero - oggetto vuoto  
if (\[\]) {           // Vero - array vuoto  
if ("ciao") {       // Vero - stringa non vuota  
if (42) {           // Vero - numero diverso da zero

if (-42) {          // Vero - numero negativo diverso da zero

*7\. Operatore di Conversione Booleana*

In JavaScript abbiamo un operatore che ci consente di trasformare un valore nel suo corrispettivo booleano. Andiamo nella console:

L'operatore doppio punto esclamativo (\!\!):

"ciao"          // "ciao" (stringa)  
\!\!"ciao"        // true (booleano)

\!\!0             // false  
\!\!10            // true  
\!\!""            // false  
\!\!" "           // true  
\!\!{}            // true  
\!\!\[\]            // true  
\!\!null          // false

\!\!undefined     // false

Il doppio punto esclamativo converte il valore nel suo booleano equivalente:

* Il primo \! inverte il valore (da truthy a false, o da falsy a true)  
* Il secondo \! ri-inverte per ottenere il booleano corretto
