Function Expressions e Callback

Lezione 33 - Corso JavaScript

*4\. Funzioni come Parametri: Callback e Higher-Order Functions*

Poiché le funzioni sono "privilegiate", possiamo passarle come argomenti ad altre funzioni esattamente come passeremmo un numero o una stringa. Questo meccanismo è alla base della programmazione funzionale e asincrona in JavaScript.

*Definizioni Chiave*

  • *Callback:* Una funzione che viene passata come parametro ad un'altra funzione. È chiamata così perché verrà "richiamata" (called back) all'interno della funzione esterna.
  • *Higher-Order Function (Funzione di Ordine Superiore):* Una funzione che riceve un'altra funzione come parametro oppure ne restituisce una.

*Esempio Pratico: Creazione di un sistema di saluti modulare*

Creiamo due funzioni base che restituiscono stringhe diverse, e una funzione di ordine superiore che gestisce l'esecuzione.

JavaScript

// 1\. Definiamo le funzioni base (che saranno usate come callback)

function ciao() {

return "Ciao";

}

function arrivederci() {

return "Arrivederci";

}

// 2\. Definiamo la Higher-Order Function 'saluta'

// Questa funzione accetta un parametro 'fn' (che ci aspettiamo sia una funzione)

function saluta(fn) {

// Invochiamo la funzione passata come parametro e stampiamo il risultato

// Qui 'fn' è la callback che viene eseguita

console.log(fn());

}

// 3\. Invocazione

// Passiamo il riferimento alla funzione 'ciao' (senza parentesi tonde\!)

saluta(ciao);

// Output: "Ciao"

// Passiamo il riferimento alla funzione 'arrivederci'

saluta(arrivederci);

// Output: "Arrivederci"

*Attenzione: Quando passiamo la callback a saluta, usiamo solo il nome (es. ciao), senza le parentesi tonde (). Se mettessimo le parentesi (ciao()), passeremmo il risultato* dell'esecuzione (la stringa), non la funzione stessa.

*Callback Anonime (Inline)*

Non è obbligatorio definire la funzione callback separatamente. Possiamo definirla direttamente all'interno delle parentesi tonde della chiamata, creando una funzione anonima "al volo".

JavaScript

// Invochiamo 'saluta' definendo la callback direttamente nell'argomento

saluta(function() {

return "Saluto definito inline\!";

});

// Output: "Saluto definito inline\!"

*Analisi dell'Esempio*

Nell'ultimo frammento di codice:

1. La funzione anonima function() { return ... } è la *Callback*.

2. La funzione saluta è la *Funzione di Ordine Superiore*.

Questo approccio ci permette di lavorare con un livello di *astrazione* più alto, separando la logica di esecuzione (dentro saluta) dalla logica specifica del compito (dentro la callback).
