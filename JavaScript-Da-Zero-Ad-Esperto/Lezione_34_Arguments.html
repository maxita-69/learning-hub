Arguments Object

Lezione 34 - Corso JavaScript

*1\. L'Oggetto arguments (ES5)*

In JavaScript, i parametri passati a una funzione sono accessibili internamente tramite un oggetto speciale chiamato *arguments*.

*Caratteristiche*

  • È un oggetto *Array-like (simile a un array): possiede una proprietà .length e permette l'accesso tramite indice (\[0\], \[1\]), ma non* possiede i metodi nativi degli array (come forEach, map, push).
  • Contiene tutti i valori passati in fase di invocazione, indipendentemente da quanti parametri siano stati definiti nella firma della funzione.

*Esempio Pratico*

Definiamo una funzione test senza dichiarare parametri espliciti e ispezioniamo arguments.

JavaScript

function test() {

console.log(arguments);

}

// Invocazione con tipi di dati misti

test(2, "Ciao", true);

*Output Console:*

JavaScript

Arguments { 0: 2, 1: "Ciao", 2: true, callee: function, length: 3, ... }

Per accedere ai singoli valori o iterare su di essi:

JavaScript

function test() {

// Accesso posizionale

console.log(arguments\[0\]); // 2

console.log(arguments\[1\]); // "Ciao"

// Iterazione con ciclo for classico (unico modo sicuro su Array-like)

for (let i = 0; i < arguments.length; i++) {

console.log(arguments\[i\]);

}

}

*2\. Discrepanza tra Definizione e Invocazione*

In JavaScript *non viene generato errore* se invochiamo una funzione passando un numero di argomenti diverso da quello definito.

*Esempio: Parametri in Eccesso*

Se passiamo più argomenti di quelli previsti, i parametri in eccesso vengono semplicemente ignorati dai parametri nominati, ma esistono comunque dentro arguments (o se non usiamo nulla, vanno persi).

JavaScript

function calcola(a, b) {

console.log(a + b);

}

// Passiamo 4 valori invece di 2

calcola(10, 20, 30, 40);

// Output: 30 (10 + 20\)

// 30 e 40 vengono ignorati dalla logica attuale

*3\. Gestione di un Numero Variabile di Argomenti*

Spesso non possiamo stabilire a priori quanti parametri riceveremo. Un caso tipico è la ricerca del valore minimo tra una serie di numeri usando Math.min().

JavaScript

// Math.min accetta una lista di argomenti separati da virgola

let minimo = Math.min(20, 10, 1, 0.5, 100);

console.log(minimo);

// Output: 0.5

Il metodo funziona con qualsiasi quantità di numeri passati. Per replicare questa flessibilità nelle nostre funzioni, ES6 ha introdotto i *Rest Parameters*.
