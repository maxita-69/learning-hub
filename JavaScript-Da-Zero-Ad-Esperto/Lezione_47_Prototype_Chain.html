# *La Catena dei Prototipi in JavaScript*

*Introduzione al Concetto di Prototipo*

In JavaScript, gli oggetti possono accedere a proprietà e metodi di altri oggetti attraverso un meccanismo definito *catena dei prototipi (prototype chain). L'obiettivo principale di questo sistema è evitare la duplicazione del codice*, permettendo agli oggetti di ereditare funzionalità da altri oggetti in modo gerarchico.

Come già sperimentato con le Constructor Function, gli oggetti creati attraverso una Constructor Function possono accedere alle proprietà e ai metodi definiti nella proprietà Prototype della Constructor Function stessa.

*Meccanismo di Ereditarietà tra Oggetti*

Consideriamo tre oggetti: OBJ1, OBJ2 e OBJ3, collegati in una relazione gerarchica:

  • *OBJ1 è il padre di OBJ2*
  • *OBJ2 è il padre di OBJ3*
  • Di conseguenza, anche *OBJ3 e OBJ1* sono in rapporto di parentela

*Distribuzione di Proprietà e Metodi*

La distribuzione delle proprietà e dei metodi lungo la catena è la seguente:

  • *OBJ1* definisce:
  • una proprietà P1
  • un metodo FN1
  • *OBJ2* definisce:
  • una proprietà P2
  • eredita sia P1 che FN1 da *OBJ1*
  • *OBJ3* definisce:
  • una proprietà P3
  • un metodo FN2
  • eredita da *OBJ2 e OBJ1* tutti i loro metodi e proprietà

Con questo meccanismo *non si verifica duplicazione del codice*: né OBJ2 né OBJ3 hanno necessità di duplicare il metodo FN1, a meno che non abbiano l'esigenza di modificarne il comportamento.

*Differenza tra Proprietà Prototype e Prototipo*

Una distinzione fondamentale, spesso fonte di confusione, riguarda la differenza tra *proprietà Prototype e prototipo*:

  • *Proprietà Prototype*: è una proprietà di nome Prototype che contiene un oggetto. È presente nelle Constructor Function.
  • *Prototipo di un oggetto A*: è l'oggetto immediatamente al di sopra di A nella catena dei prototipi, da cui A eredita proprietà e metodi.

*Esempio Pratico con Oggetti Letterali*

*Creazione dell'Oggetto Base*

Definiamo un oggetto letterale Ferrari con le seguenti proprietà:

const Ferrari = {

Marchio: "Ferrari",

Dull: "dato casuale",

Colore: "rosso"

};

*Creazione di un Oggetto con Prototipo Specificato*

Per creare un oggetto il cui prototipo sia l'oggetto Ferrari, utilizziamo il metodo *Object.create()*, passando in input l'oggetto che vogliamo sia prototipo del nuovo oggetto:

const Ferrari458 = Object.create(Ferrari);

In questo caso, vogliamo che il prototipo dell'oggetto Ferrari458 sia l'oggetto Ferrari.

*Verifica del Comportamento*

console.log(Ferrari458);

// Output: {} (oggetto vuoto)

console.log(Ferrari458.Marchio);

// Output: "Ferrari"

Nonostante Ferrari458 sia un oggetto vuoto, riusciamo ad accedere alla proprietà Marchio dell'oggetto Ferrari grazie al meccanismo della catena dei prototipi.

*Definizione di Proprietà sull'Oggetto Derivato*

Definiamo alcune proprietà specifiche sull'oggetto Ferrari458:

Ferrari458.Cilindrata = "4500cc";

Ferrari458.Cavalli = 570;

Ferrari458.Colore = "rosso opaco";

console.log(Ferrari458.Colore);

// Output: "rosso opaco" (valore definito su Ferrari458)

Se commentiamo la riga che definisce la proprietà Colore su Ferrari458:

Ferrari458.Cilindrata = "4500cc";

Ferrari458.Cavalli = 570;

// Ferrari458.Colore = "rosso opaco"; // riga commentata

console.log(Ferrari458.Colore);

// Output: "rosso" (valore ereditato da Ferrari)

*Meccanismo di Risoluzione delle Proprietà*

Come si comporta il motore JavaScript nella risoluzione delle proprietà?

1. *Ricerca locale*: Il motore JavaScript cerca innanzitutto la proprietà all'interno dell'oggetto Ferrari458

2. *Risalita della catena*: Se non la trova, inizia a risalire la catena dei prototipi

3. *Ricerca nel prototipo*: Va a ricercare la proprietà sull'oggetto Ferrari

4. *Restituzione del valore*: Trova la proprietà e restituisce il valore (in questo caso "rosso")

*Estensione della Catena dei Prototipi*

Una catena solitamente lega più di due elementi, e infatti possiamo allungarla ulteriormente. Possiamo creare un oggetto che abbia come prototipo l'oggetto Ferrari458:

const Ferrari458Special = Object.create(Ferrari458);

console.log(Ferrari458Special);

// Output: {} (oggetto vuoto)

*Definizione di Proprietà e Ricerca Multi-Livello*

Definiamo una proprietà specifica:

Ferrari458Special.Colore = "nero";

console.log(Ferrari458Special.Colore);

// Output: "nero"

Il motore JavaScript ricerca la proprietà Colore all'interno dell'oggetto Ferrari458Special, la trova e quindi restituisce il valore.

*Primo caso* - Commentiamo la definizione locale:

// Ferrari458Special.Colore = "nero"; // riga commentata

console.log(Ferrari458Special.Colore);

// Output: "rosso opaco"
